Để tạo một trang web, bạn cần làm theo các bước sau:

1. Tạo một tệp HTML

Bạn có thể sử dụng Notepad, VS Code, Sublime Text, hoặc bất kỳ trình soạn thảo nào để tạo tệp .html.

Ví dụ:

  1. Mở Notepad (hoặc VS Code).
  2. Sao chép mã HTML mà tôi đã viết ở trên.
  3. Lưu lại với tên: index.html (Chọn All Files khi lưu).

2. Mở trang web trên trình duyệt

  • Sau khi lưu tệp index.html, nhấp đúp vào tệp đó để mở bằng trình duyệt như Chrome, Firefox, Edge.

3. Nếu muốn đưa lên mạng (Hosting trang web)

Có nhiều cách để đưa trang web của bạn lên Internet:

  1. Dùng GitHub Pages (Miễn phí, đơn giản).
  2. Dùng Netlify / Vercel (Nhanh chóng, miễn phí).
  3. Mua Hosting + Tên miền nếu muốn chuyên nghiệp hơn.

Dưới đây là hướng dẫn từng bước để đưa trang web của bạn lên GitHub Pages miễn phí:

Bước 1: Tạo tài khoản GitHub

  1. Truy cập GitHub và đăng ký (hoặc đăng nhập nếu bạn đã có tài khoản).
  2. Sau khi đăng nhập, bạn sẽ thấy giao diện GitHub.

Bước 2: Tạo Repository (Kho lưu trữ)

  1. Nhấn vào nút "+" ở góc phải trên cùng → Chọn New repository.
  2. Đặt tên repository (VD: my-website).
  3. Chọn Public (Công khai).
  4. Chọn Add a README file (không bắt buộc).
  5. Nhấn Create repository.

Bước 3: Tải mã nguồn lên GitHub

Bạn có thể làm theo 1 trong 2 cách sau:

Cách 1: Tải lên trực tiếp trên GitHub

  1. Trong repository vừa tạo, nhấn Add file → Upload files.
  2. Kéo và thả tệp index.html (và các tệp khác như style.css nếu có).
  3. Nhấn Commit changes để lưu lại.

Cách 2: Dùng Git (Dành cho người có Git trên máy)

  1. Cài đặt Git (Download Git) nếu bạn chưa có.
  2. Mở Git Bash và nhập lệnh sau:
sh
git clone https://github.com/TenCuaBan/my-website.git
cd my-website
  1. Sao chép tệp index.html vào thư mục này, sau đó nhập lệnh:
sh
git add .
git commit -m "Upload website"
git push origin main

Bước 4: Kích hoạt GitHub Pages

  1. Vào repository của bạn trên GitHub.
  2. Nhấp vào Settings (Cài đặt).
  3. Trong thanh bên trái, chọn Pages.
  4. Ở phần Source, chọn Deploy from a branch.
  5. Ở Branch, chọn main hoặc master rồi nhấn Save.
  6. Chờ 1-2 phút, GitHub sẽ tạo link:
    👉 https://tencuaban.github.io/my-website/

Xong! 🎉
